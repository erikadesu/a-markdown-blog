# Bem vindo ao Melhor e Mais Simples Blog do Mundo! 🤩

Aqui você pode postar seus pensamentos e ideias mais mirabolantes de forma simples e rápida em **Markdown**, uma linguagem de formatação muito simples, eventualmente convertida para HTML.

## Como funciona? 

* Basta criar uma conta e começar a postar! ✍️
* Seu post será convertido para HTML em pouquísismos instantes pelos nossos gnomos mágicos do **Ruby on Rails**! 🧙‍♂️
* Você poderá checar o status da conversão dos seus posts no seu dashboard __My Posts__ 
* Você também poderá editar e apagar seus posts 👻
* Todos os posts convertidos para HTML aparecerão na homepage para todo mundo ler ☺️
